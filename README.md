# Airbnb Listings Analysis – Chicago & New Orleans

## Project Objective
The objective of this project is to analyze Airbnb listings data from Chicago and New Orleans using data visualization techniques.

## Tools Used
- Tableau
- Data Cleaning
- Data Visualization

## Dashboard Insights
The dashboard provides insights into:
- Total Airbnb Listings
- Average Listing Price
- Total Reviews
- Price comparison by room type
- Top neighborhoods with the most listings
- Geographic distribution of listings

## Key Findings
1. Hotel rooms have the highest average price.
2. Entire homes are the second most expensive accommodation type.
3. Central Business District has the highest number of listings.
4. Airbnb activity is concentrated in high-traffic neighborhoods.

## Dashboard
The dashboard includes:
- KPI Metrics
- Map Visualization
- Price by Room Type
- Top Neighborhood Analysis
